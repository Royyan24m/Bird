# Bird Game

A simple and fun flying bird game built with HTML, CSS, and JavaScript. The objective is to navigate the bird through a series of pipes without colliding.

## Features

-   **Simple Controls:** Use the `spacebar` or `click` to make the bird fly.
-   **Dynamic Gameplay:** Pipes are generated at random heights, making each playthrough unique.
-   **Score Tracking:** Your score increases for each pipe you successfully pass.
-   **Responsive Design:** The game canvas adjusts to the size of your browser window.

## How to Play

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Royyan24m/Bird.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Bird
    ```
3.  **Open the `src/index.html` file in your browser.**
4.  **Click the "Mulai" (Start) button to begin the game.**

## Project Structure

The project is organized into the following directories:

-   `src/`: Contains the core source code for the game.
    -   `index.html`: The main HTML file.
    -   `style.css`: The stylesheet for the game.
    -   `script.js`: The JavaScript file containing the game logic.
-   `assets/`: Contains the image assets used in the game.
